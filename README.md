## Criptografia AES-128 (Advanced Encryption Standard)

 Bloco de aceleração para algoritmo de criptografia.
Esse algoritmo já é amplamente utilizada em processadores e  é essencial para criptografia de dados em disco rígido e comunicação em redes, portanto torna-se necessário que seja feito o mais eficiente possível.

AES é um algoritmo de criptografia simétrica


 Para executar o código python em aes_python

 ```bash
cd aes_python/ && python3 -m venv venv && source venv/bin/activate && pip install cryptography && python3 aes_sample.py
 ```

Para executar o código cpp em aes_cpp

```bash
cd aes_cpp/ && make run && make test
```
